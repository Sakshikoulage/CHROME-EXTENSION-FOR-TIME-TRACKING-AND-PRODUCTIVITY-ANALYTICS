This project is a full-stack productivity tracking system built using a Chrome Extension, Node.js backend, MongoDB database, and a React dashboard.

The Chrome extension tracks the time users spend on different websites and classifies them as Productive, Unproductive, or Neutral. The backend processes and stores this data, while the React dashboard visualizes productivity analytics with charts and weekly reports.

This project demonstrates real-world full-stack development including browser APIs, REST APIs, database design, analytics logic, and frontend data visualization.

Tracks time spent on active browser tabs

 Classifies websites (Productive / Unproductive / Neutral)

 Weekly productivity calculation (last 7 days)

 Pie chart for usage distribution

Daily productivity bar chart

 Weekly productivity percentage score

Modern dark-themed dashboard UI
